Bienvenue 👋

Site de révision et d’outils pour les élèves (collège). Utilise le menu ci‑dessus pour accéder aux outils et aux documents.

  • Glisse‑nombre pour entraîner les multiplications/divisions par 10, 100, 1000.
  • Dossier Documents pour télécharger les fiches PDF.

Mettre en ligne un PDF

  1. Dépose ton fichier dans le dossier docs/.
  2. Ajoute un lien dans cette page (ou laisse les élèves ouvrir docs/).

Astuce : renomme tes fichiers clairement (ex. fiche-division-6e.pdf).
