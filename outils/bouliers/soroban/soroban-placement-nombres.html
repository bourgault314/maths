Soroban interactif — entraînement

Cible : —
Gestes : unaires = clic (la 4ᵉ ⇒ 4 actives ; à 3 actives, clic 1ʳᵉ ⇒ 0 ; clic 2ᵉ ⇒ 2, etc.). Quinaire = clic (active/désactive). Couleurs : Shift/Ctrl/Alt + clic (ou appui long) → blanc → gris → gris foncé. Validation : la vérification est automatique et passe au suivant dès que c’est correct.
