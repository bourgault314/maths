🛠️ Instructions d'assemblage

  1. Imprimez le modèle. L'option "Version XL" agrandit la hauteur pour faciliter l'écriture !
  2. Le Tableau : Découpez en suivant les pointillés gris (conservez la marge pour écrire).
  3. Le Curseur : Découpez le contour bleu extérieur. Avec un cutter, évidez le rectangle blanc central (gardez les fins montants colorés).
  4. Fentes et assemblage : Plastifiez vos éléments. Incisez des fentes dans le plastique du curseur pour faire passer la bande derrière la fenêtre et devant les montants.
