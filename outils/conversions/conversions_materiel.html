🛠️ Instructions de fabrication

  1. Impression : Imprimez ce document sur une feuille A4 épaisse. (Cochez "Version XL" si vous souhaitez de plus grandes cases pour écrire).
  2. Le Tableau : Découpez en suivant le grand cadre en pointillés gris. Conservez bien la marge blanche pour écrire les grands nombres.
  3. Le Curseur : Découpez le contour bleu extérieur. Avec un cutter, évidez le grand rectangle blanc central (en gardant les fines bordures bleues intactes).
  4. Assemblage : Plastifiez les deux éléments séparément. Faites deux fentes verticales dans le plastique du curseur pour faire glisser la grande bande dedans.
