Aide rapide

Raccourcis & usage en classe (projection).

Créer
  • + : ajouter +1
  • - : ajouter -1
  • Suppr : effacer tout
Actions
  • Espace : trier
  • Entrée : auto 0
  • Ctrl+Z : annuler
Outils
  • Main : déplacer les jetons
  • Stylo : annoter (croix)
  • Son : on/off
Projection
  • F : plein écran
  • Mode projection : interface simplifiée
  • Ctrl + molette : zoom (recommandé 125–150%)
  • H : afficher l’aide
Astuce : tu peux laisser le son off en classe et l’activer en démo si tu veux.
